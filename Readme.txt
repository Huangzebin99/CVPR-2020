算法主要思想主要在一下两点：

数据/模型处理：
1.）数据增广：
    一、将真人ir数据添加 亮斑 作为攻击；
    二、将真人的rgb图片转为灰度格式，并加入 亮斑、模糊 等效果，作为攻击

2.）模型训练：
    使用小网络训练

图像处理方法：
    一、通过计算 深度图 的整体亮度，非0像素的方差等，过滤一部分攻击，并通过一部分真人；
    二、通过轮廓检测提取 IR图上可能存在亮斑，过滤一部分攻击；
    三、对于图像处理方法难以分辨的样本，通过深度学习模型进行判断，并对一个id下所有数据进行投票，给出最终结果（深度模型先进行投票，如果判断为真人，再进行ir模型的投票，以ir模型结果为最终结果；如深度模型判断为攻击，则最终结果判断为攻击）
